# Welcome to LoopBACK.[^1]
A peer-review network for students to trade project feedback powered by a credit loop economy.

## Main features
- **Real-time Feed & Smart Search:** Dynamic list of active projects with categorization (UX/UI, Frontend, Backend, etc.) and complexity filtering, fully synced in real-time with Firestore.
- **Visual Canvas Annotation:** Click-to-pin canvas implementation that allows precise, numbered, and contextual critique overlays directly on project images.
- **Credit Loop Economy:** A secure transactional wallet logic that auto-subtracts credits upon publishing and auto-rewards upon uality review completion.
- **Anti-Spam Validation:** Smart input metrics enforcing character limits (minimum 30 characters combined) to block fast-click rewards and protect feedback quality.
- **Mutual Peer Connections (Networking):** Real-time social layer where users can follow others, changing statuses from "Members" to "Peers" once a follow-back occurs.
- **File & Image Uploads:** Live cloud storage management utilizing Firebase Storage buckets for cover and high-fidelity project files.

## Who is it for?
LoopBACK. is strictly designed as a safe, beginner-friendly community for students. It targets early-stage developers, designers, and creators who need constructive feedback but often feel intimidated by mainstream, highly competitive tech platforms. By connecting peers who are in the exact same learning stage, the ecosystem fosters empathy, reduces the fear of judgment, and encourages horizontal collaboration among peers navigating the same educational hurdles.
